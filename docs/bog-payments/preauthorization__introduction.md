Introduction
These methods allow businesses to complete the withdrawal process for the amount that was previously blocked during the order request These methods are useful if the value of the capture field was set to manual during the order request. There are three methods available for completing the payment: full confirmation of the amount, partial confirmation of the amount, and rejection of the pre-authorization.
